How to deploy as a live website (free)
1

Create a free Firebase project

Go to console.firebase.google.com → Add project → name it ipl-draft-2026 → disable Google Analytics → Create project.

2

Enable Realtime Database

Sidebar → Build → Realtime Database → Create database → choose a region → Start in test mode. This allows read/write for 30 days — more than enough for your draft.

3

Get your Firebase config

Project Settings (gear icon, top-left) → General → scroll to "Your apps" → click the web icon </> → Register app → copy the firebaseConfig object shown.

4

Paste config into the HTML file

Open IPL2026_Draft_Firebase.html in Notepad++ or VS Code → find the FIREBASE_CONFIG block at the very top → replace each PASTE_YOUR_..._HERE with your real values → save the file.

5

Host on Netlify — 2 minutes, free forever

Go to netlify.com → Sign up (free) → Sites → "Add new site" → "Deploy manually" → drag & drop the HTML file into the box → Netlify gives you a URL like amazing-draft-a1b2.netlify.app.

Share that URL with all 10 team owners. Done!

6

Alternative: GitHub Pages (also free)

Create a GitHub account → New repository → name it ipl-draft → upload the HTML file renamed to index.html → Settings → Pages → Source: main → Your URL will be yourusername.github.io/ipl-draft.

Security note

Firebase test mode lets anyone with the URL read/write. For a private league draft this is perfectly fine — only your group knows the link. The 30-day test window will outlast your draft. After that, update the database rules in Firebase console if needed.
